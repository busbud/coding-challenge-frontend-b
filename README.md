
## Run locally
```
export REACT_APP_API_TOKEN='your_secret_busbud_api_access_code'
git clone git@github.com:iamseye/coding-challenge-frontend-b.git
cd coding-challenge-frontend-b
npm install
```
To boot the server: `npm start`.
